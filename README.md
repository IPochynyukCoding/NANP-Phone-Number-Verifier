# North American Number Plan (NANP) Phone Number Verifier
## Description
An implementation in Python to verify North American Number Plan (NANP) phone numbers using regular expressions in a JSON file to provide stricter tolerances than non-regular-expression methods such as removing parentheses and dashes. This program is made for the COSC 1336 class as an alternative project.

## How to use
1. Enter your phone number
2. Wait for the program to display your results
